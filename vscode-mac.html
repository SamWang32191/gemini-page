VS Code Shortcuts
Mac Edition

掌握快捷鍵，開發更神速

一般操作

使用者設定 ⌘ ,
快速開啟檔案 ⌘ P
顯示指令面板 ⇧ ⌘ P
快捷鍵設定視窗 ⌘ K ⌘ S
關閉當前檔案 ⌘ W
關閉視窗 ⇧ ⌘ W

基本編輯

剪下此行 (未選取時) ⌘ X
複製此行 (未選取時) ⌘ C
復原 ⌘ Z
取消復原 (重做) ⇧ ⌘ Z
刪除此行 ⇧ ⌘ K
向上插入一行 ⇧ ⌘ Enter
向下插入一行 ⌘ Enter
向上/下移動此行 ⌥ ↑ / ↓
向上/下複製此行 ⌥ ⇧ ↑ / ↓
切換行註解 ⌘ /
格式化文件 ⇧ ⌥ F
快速修復 / 顯示建議 ⌘ .

選取與多游標

加入下一個相同結果 ⌘ D
選擇所有匹配項 ⇧ ⌘ L
展開選取範圍 ⌃ ⇧ →
縮小選取範圍 ⌃ ⇧ ←
選取目前行 ⌘ L
全選 ⌘ A

搜尋與取代

在檔案中尋找 ⌘ F
在檔案中取代 ⌥ ⌘ F
在所有檔案中尋找 ⇧ ⌘ F
在所有檔案中取代 ⇧ ⌘ H

導航與檔案管理

前往檔案 ⌘ P
前往符號 (函式/變數) ⇧ ⌘ O
前往行號 ⌃ G
在分頁間切換 ⌃ Tab
關閉所有分頁 ⌘ K W
關閉其他分頁 ⌥ ⌘ W
回到上一個游標位置 ⌃ -
前往下一個游標位置 ⌃ ⇧ -

檢視與側邊欄

顯示/隱藏 Explorer ⇧ ⌘ E
顯示搜尋面板 ⇧ ⌘ F
顯示 Source Control ⌃ ⇧ G
顯示 Debug 面板 ⇧ ⌘ D
顯示 Extensions 面板 ⇧ ⌘ X
Zen Mode (禪模式) ⌘ K Z

編輯器與終端

分割編輯器 ⌘ \
在群組間切換 ⌘ 1
開啟內建終端 ⌃ `
新建終端 ⌃ ⇧ `
切換終端 ⌃ ⌥ →

小撇步

如果想看完整官方快捷鍵清單，可以在 VS Code 中按 ⌘ K ⌘ R 叫出 PDF，或直接參考 官方 macOS 快捷鍵表格。
