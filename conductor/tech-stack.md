# Tech Stack - gemini-page

## 核心運行環境 (Runtime & Platform)
- **Web Browser**: 專案部署於 GitHub Pages，直接於瀏覽器運行，不需後端服務。

## 前端技術 (Frontend Technologies)
- **Vanilla HTML5**: 用於定義結構化的內容與語義化標籤。
- **Vanilla CSS3**: 負責頁面的視覺設計、卡片式佈局與互動動畫。
- **Vanilla JavaScript (ES6+)**:
  - 用於處理技術模擬器 (Labs) 的互動邏輯。
  - 管理頁面狀態與數據綁定。

## 內容管理與工作流 (Content & Workflow)
- **JSON Metadata (`pages.json`)**: 作為專案的數據庫，管理所有技術頁面的清單、標籤、顏色與元數據。
- **AI-Powered Pipeline**:
  - **Gemini (Deep Research)**: 獲取高準確度的技術知識與研究數據。
  - **Canvas**: 將研究成果轉化為高品質、具備互動性的前端 HTML 代碼。

## 工具與開發環境 (Tools)
- **Git**: 版本控制。
- **Conductor**: 專案開發流程與 AI 代理協作框架。
