# Product Guide - gemini-page

## 產品願景 (Vision)
作為一個專為 **全端工程師 (Full-stack Engineer)** 打造的現代化技術知識庫與學習平台。透過 Gemini 的深度研究 (Deep Research) 能力獲取精確知識，並利用 Canvas 產生高品質的 HTML 互動內容，提供從前端、後端到 DevOps 的全方位技術導航。

## 目標對象 (Target Audience)
- **全端工程師：** 跨足 Frontend、Backend 與 DevOps 領域，需要快速掌握跨領域知識的開發者。
- **SRE 與運維專家：** 關注 K8s、Linux 權限與基礎設施實踐。
- **追求高效學習者：** 偏好透過互動式模擬器（Labs）而非純文字文件來理解複雜邏輯。

## 核心價值 (Core Values)
1. **AI 驅動的準確性：** 所有技術內容皆經過 Gemini Deep Research 驗證，確保資訊的即時性與權威性。
2. **互動式學習 (Learning by Doing)：** 不僅是閱讀文件，更提供模擬器讓使用者在預覽環境中實驗技術細節（如 Linux chmod, K8s NFS 權限）。
3. **美學與效能：** 網頁設計美觀且響應迅速，作為現代化靜態技術文件的示範。

## 主要功能 (Key Features)
- **多維度技術實作 (Tech Interactive Labs)：** 提供互動式模擬器，將抽象概念（如 Git Rebase, K8s RBAC）具象化。
- **雙平台快捷鍵導航 (Multi-platform Shortcuts)：** 支援在 macOS 與 Windows 鍵位之間即時切換，滿足不同作業系統開發者的需求。
- **全端技術索引 (Full-stack Tech Index)：** 整理彙整包括 Go, Kubernetes, Helm, Frontend Frameworks 等領域的速查表與深度指南。
- **AI 自動化內容流水線：** 標準化「Gemini 研究 -> Canvas 生成 -> HTML 部署」的流程，快速迭代高品質內容。
