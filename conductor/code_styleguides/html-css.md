# Google HTML/CSS 風格指南摘要

本文件總結了 Google HTML/CSS 風格指南中的關鍵規則與最佳實踐。

## 1. 一般規則
- **通訊協定：** 所有嵌入資源皆使用 HTTPS。
- **縮排：** 縮排 2 個空格。不要使用 tab。
- **大小寫：** 程式碼 (元素名稱、屬性、選擇器、屬性值) 一律使用小寫。
- **結尾空白：** 移除所有行尾空白。
- **編碼：** 使用 UTF-8 (不含 BOM)。在 HTML 中指定 `<meta charset="utf-8">`。

## 2. HTML 風格規則
- **文件型別 (Document Type)：** 使用 `<!doctype html>`。
- **HTML 有效性：** 使用有效的 HTML。
- **語義化 (Semantics)：** 根據用途使用 HTML 元素 (例如：使用 `<p>` 表示段落，而不是為了留白)。
- **多媒體備援 (Fallback)：** 為圖片提供 `alt` 文字，為音訊/影片提供逐字稿或字幕。
- **關注點分離：** 嚴格分離結構 (HTML)、表現 (CSS) 和行為 (JavaScript)。連結到外部 CSS 和 JS 檔案。
- **`type` 屬性：** 省略樣式表 (`<link>`) 和腳本 (`<script>`) 的 `type` 屬性。

## 3. HTML 格式化規則
- **一般：** 每個區塊、列表或表格元素都另起新行，並對其子元素進行縮排。
- **引號：** 屬性值使用雙引號 (`""`)。

## 4. CSS 風格規則
- **CSS 有效性：** 使用有效的 CSS。
- **類別命名 (Class Naming)：** 使用有意義、通用的名稱。單字之間以連字號 (`-`) 分隔。
  - **優：** `.video-player`, `.site-navigation`
  - **劣：** `.vid`, `.red-text`
- **ID 選擇器：** 避免使用 ID 選擇器來定義樣式。優先使用類別選擇器。
- **簡寫屬性 (Shorthand Properties)：** 盡可能使用簡寫屬性 (例如：`padding`、`font`)。
- **`0` 與單位：** `0` 值省略單位 (例如：`margin: 0;`)。
- **前導 `0`：** 小數值應包含前導 `0` (例如：`font-size: 0.8em;`)。
- **十六進位表示法：** 盡可能使用 3 個字元的十六進位表示法 (例如：`#fff`)。
- **`!important`：** 避免使用 `!important`。

## 5. CSS 格式化規則
- **宣告順序：** 規則內的宣告應按字母順序排列。
- **縮排：** 對所有區塊內容進行縮排。
- **分號：** 每個宣告後都使用分號。
- **空格：**
  - 屬性名稱的冒號後使用空格 (`font-weight: bold;`)。
  - 最後一個選擇器與左大括號之間使用空格 (`.foo {`)。
  - 每個選擇器和宣告皆另起新行。
- **規則分隔：** 規則之間以新行分隔。
- **引號：** 屬性選擇器和屬性值使用單引號 (`''`) (例如：`[type='text']`)。

**保持一致性。** 修改程式碼時，請配合現有風格。

*來源：[Google HTML/CSS Style Guide](https://google.github.io/styleguide/htmlcssguide.html)*
